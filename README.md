> **The Superheroes App** is a modern Android application built using **Jetpack Compose**. It displays a scrollable list of superhero profiles with images, names, and descriptions using **Material Design 3** components. <br><br> This project demonstrates *clean architecture, reusable composables, custom theming, and structured UI design.*

<br>

By building this project, I have mastered:
* 🧱 **How to structure a Compose project** for scalability.
* 🔗 **How to separate UI and data layers** effectively.
* 📜 **How to implement scrollable lists** using Lazy layouts.
* 🎨 **How to use Material Design 3 components** properly for a native feel.
